# Scroll Slide

Simple slide show using only the mouse-wheel/touchpad scroll.


## Usage

```html

<div class="ss-container"> 
  <div class="ss-item active">
    <!-- content... -->
  </div>
  <div class="ss-item">
    <!-- content... -->
  </div>
  <!-- Add as many ss-items as you need. -->
</div>

```
```javascript
  //Initialize
  scrollSlide('.ss-container', '.ss-item');

```
